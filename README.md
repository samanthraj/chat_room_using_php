💬 PHP MySQL Chatroom Application

A real-time chatroom web application built using PHP and MySQL that allows users to create accounts, join chatrooms, and communicate with each other seamlessly.

This project demonstrates core concepts of user authentication, database management, session handling, and dynamic message exchange in a multi-user environment.

🚀 Features

User registration and login system

Create and join multiple chatrooms

Real-time message exchange between users

Persistent message storage using MySQL

Session-based authentication

Clean and simple UI for interaction

Organized backend structure using PHP

🛠️ Tech Stack

Frontend: HTML, CSS

Backend: PHP

Database: MySQL

📚 What This Project Demonstrates

Handling user authentication and sessions

Designing relational databases for chat systems

CRUD operations with PHP and MySQL

Building multi-user interactive web applications
